tr: table row 表示表格的一行
td: table data 表示表格的数据单元格
th: table header 表示表格的表头单元格
thead: 用于定义表格的标题部分, 在thead中使用 "th" 来定义列的标题
tbody: 用于定义表格的主体部分, 在tbody中使用 "tr" 来定义行, 在每行中使用 "td" 来定义单元格数据
