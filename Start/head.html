head 元素包含了所有的头部标签元素。在 head元素中你可以插入脚本(scripts), 样式文件(CSS),及各种meta信息。可以添加在头部区域的元素标签为: title, style, meta, link, script, noscript 和 base。

<link> tag usually used to connect to the CSS sheet here is an example

<link rel="stylesheet" type="text/css" href="mystyle.css">

我们也可以直接用 style 标签, 并在其中直接添加样式来渲染HTML文档

	<style type="text/css">
		body {
			background-color:yellow;
		}
		p {
			color:blue
		}
	</style>
